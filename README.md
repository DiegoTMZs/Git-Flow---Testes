# docs: comandos principais de Git

Clonar um repositório:
git clone [repositorio]

Iniciar o git no seu projeto:
git init

Preparar um arquivo para salvamento:
git add [arquivo]

Salva alterações com uma mensagem:
git commit -m "[texto]"

Envia as alterações locais para o repositório em núvem:
git push

Configura o nome do usuário do git:
git git config --global user.name "[Seu Nome]"

Configura o email do usuário do git:
git config --global user.email "[email]"

Informa os status do projeto atual:
git status

Lista todas as branch existentes:
git branch

Cria e muda para uma nova branch:
git checkout -b [branch]

Mudar de uma brach para outra
git checkout [branch]

Combina as alterações com uma branch existente:
git merge [branch]
